This plugin will add the site's top-level pages to the BuddyPress main navigation

== Description ==

This very simple plugin adds all top-level static pages to the main Buddypress Navigation. This is great for adding a "Contact Us" or "About" page in a seamless manner.

Please email me with any feedback.  peter@studio66design.com
 
== Installation ==

Make sure WPMU and BuddyPress are installed and active.

Copy the plugin folder buddypress-pages-to-navigation/ into /wp-content/plugins/

Browse to the plugin administration screen and activate the plugin.

Please don't hesitate to contact me, especially if you run into trouble. peter@studio66design.com
